**HOW TO CLONE THE PROJECT**
- [ON DESKTOP]
- open github desktop
- select artshowcase
- click clone button


**HOW TO RUN NEWLY CLONED PROJECT**
- [ON TERMINAL]
- open terminal
- cd frontend
- npm install
- cd ..
- cd backend
- npm install


**HOW TO PULL UPDATED VERSION**
- [ON DESKTOP]
- open github desktop
- click "pull" button


**HOW TO PUSH CERTAIN UPDATED FILES ONLY**
- [ON TERMINAL]
- open terminal
- git add (filename)
- git commit -m "base files"
- [ON DESKTOP]
- open github desktop
- click push origin


**HOW TO PUSH ALL UPDATED FILES**
- [ON TERMINAL]
- open terminal
- git add .
- git commit -m "base files"
- [ON DESKTOP]
- open github desktop
- click push origin
