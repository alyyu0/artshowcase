**HOW TO PULL UPDATED VERSION**
[ON DESKTOP]
- open github desktop
- click "pull" button


**HOW TO PUSH CERTAIN UPDATED FILES ONLY**
[ON TERMINAL]
- git add (filename)
- git commit -m "base files"
[ON DESKTOP]
- open github desktop
- click push origin


**HOW TO PUSH ALL UPDATED FILES**
[ON TERMINAL]
- git add .
- git commit -m "base files"
[ON DESKTOP]
- open github desktop
- click push origin
