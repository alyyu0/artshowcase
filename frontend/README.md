**INSTALL COMMANDS**
npm install react-bootstrap bootstrap
npm install lucide-react
npm install react-bootstrap-icons


**TO CHECK PROGRESS IN LOCALHOST**
IN TERMINAL, INPUT:
1. cd frontend
2. npm run dev


**TO UPLOAD MODIFIED FILES IN GITHUB**
IN TERMINAL, INPUT:
(try git status to be sure of the files)
1. git add [filename]
(example: git add Signup.jsx App.css App.jsx)
2. git commit -m "base files"
3. git branch -M main
4. git push -u origin main


**HOW TO CLONE THE PROJECT**
1. git clone https://github.com/alyyu0/artshowcase.git
2. cd frontend
3. npm install
4. npm run dev
